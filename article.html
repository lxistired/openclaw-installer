国内用户面临的四个障碍

1
网络限制 — OpenClaw 默认连接海外服务器，国内网络无法直连
2
环境依赖 — 需要预装 Node.js ≥ 22、Git 并配置 npm，对非开发者而言操作陌生
3
API 配置 — 需要注册 AI 平台账号、获取 API Key 并手动设置环境变量，流程繁琐
4
消息平台 — 国外的 Telegram 等平台国内受限，无法作为日常 AI 入口

OpenClaw 是什么

OpenClaw 是一个开源的 AI 助手框架，通过接入大语言模型，让你在命令行或飞书中直接与 AI 对话、完成各种任务。与传统的 AI 聊天网页不同，OpenClaw 能够直接操作你的文件和项目——写代码、处理数据、生成文档，真正做到"AI 帮你干活"。

它可以帮你完成：

✓代码编写与调试——用自然语言描述需求，AI 直接生成代码

✓数据处理与分析——批量清洗、转换、汇总数据

✓文档生成与处理——自动撰写报告、提取关键信息

✓飞书机器人——在群聊或私聊中 @机器人 即可使用 AI

双窗口引导，小白也能用

安装过程中的每一步配置（注册 AI 平台、获取 API Key、创建飞书应用），脚本都会同时打开两个窗口：

1
浏览器窗口 A — 本地图文教程

你只需要在教程和平台之间切换，照着做就行。完成后回到命令行窗口粘贴 API Key，脚本自动验证并写入配置。

六步安装流程

步骤 1/6 安装 Node.js

自动检测系统是否已安装 Node.js ≥ 22。未安装则从国内镜像（npmmirror.com）下载安装，全程无需翻墙。

步骤 2/6 安装 Git

自动检测 Git 是否已安装。未安装则从国内镜像下载安装。

步骤 3/6 安装 OpenClaw

配置 npm 国内镜像源，然后通过 npm 全局安装 OpenClaw。

步骤 4/6 配置 AI 提供商

选择智谱 GLM、Kimi 或 MiniMax，脚本自动打开图文教程和平台注册页。获取 API Key 后粘贴到命令行，脚本自动验证连接并写入配置。

步骤 5/6 配置飞书机器人（可选）

选择是否启用飞书机器人。启用则自动打开飞书教程，引导创建应用、获取 App ID/Secret、配置权限。个人飞书账号即可操作，无需企业认证。

步骤 6/6 生成配置并启动

脚本自动将所有配置写入本地文件，创建桌面快捷方式，并启动 OpenClaw。

常见问题

Q使用费用如何？

智谱提供免费额度，其中 glm-4-flash 模型完全免费。MiniMax 价格最低（$0.15/M input），适合大量使用。Kimi 以 256K 超长上下文见长。安装时可根据需求自由选择。

Q安全性如何保证？

安装脚本完全开源，所有代码可在 GitHub 查看。脚本仅安装官方依赖并写入本地配置文件，不收集任何用户信息，API Key 仅保存在你自己的电脑上。

Q飞书机器人需要企业账号吗？

不需要。个人飞书账号登录开放平台后，如果还没有团队，系统会引导你创建一个——随便填个团队名称（如「我的工作室」）即可，无需企业认证。

Q安装后可以切换 AI 提供商吗？

可以。运行「配置API.bat」即可随时切换提供商、更换模型、更新 API Key。

Q系统有什么要求？

Windows 10 及以上即可。安装过程需要管理员权限（右键「以管理员身份运行」）。安装完成后正常使用无需管理员权限。
