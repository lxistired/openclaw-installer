Kimi / 月之暗面 — API Key 获取教程

256K 超长上下文，kimi-k2.5 旗舰 1T MoE

1

注册月之暗面开放平台账号

在另一个浏览器窗口中，打开 Kimi 开放平台：

https://platform.moonshot.ai

点击 「注册」，使用手机号或邮箱完成注册。

已有账号则直接 「登录」。

2

实名认证（如需）

如果系统提示需要实名认证：

1. 进入「账户设置」

2. 按提示完成手机号验证和实名认证

新用户注册后通常会赠送免费额度，可以直接试用。
3

进入 API Key 管理页面

登录后，直接打开 API Key 管理页面：

https://platform.moonshot.ai/console/api-keys

或者通过导航：控制台 → 「API Key 管理」

4

创建 API Key

在 API Key 管理页面：

1. 点击 「新建」 或 「创建 API Key」

2. 输入名称（如「OpenClaw」）

3. 点击 「确认」

API Key 只显示一次！创建后请立即复制保存。关闭弹窗后无法再次查看。
5

复制 API Key

创建成功后，点击 「复制」 按钮复制 API Key。

Kimi API Key 格式类似：

sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
建议把 API Key 临时保存到记事本中，方便稍后粘贴。
6

回到安装程序

复制好 API Key 后：

1. 切换回安装程序窗口（命令行黑色窗口）

2. 在提示处粘贴 API Key（右键粘贴或 Ctrl+V）

3. 按 Enter 确认

安装程序会自动测试 API 连接是否正常。

❓ 常见问题

Q: API Key 丢失了怎么办？

A: API Key 只在创建时显示一次。请删除旧 Key 后重新创建。


Q: 调用提示 401 错误？

A: API Key 可能已失效或被删除。请在控制台确认 Key 状态，或重新创建。


Q: 使用国内端点还是国际端点？

A: 在中国大陆使用 api.moonshot.cn（国内端点），安装程序已自动配置。
