智谱 GLM — API Key 获取教程

国产最强大模型，GLM-5 旗舰 744B MoE

1

注册智谱开放平台账号

在另一个浏览器窗口中，打开智谱开放平台：

https://open.bigmodel.cn

点击右上角 「注册」，使用手机号完成注册。

已有账号则直接 「登录」（支持微信扫码登录）。

2

实名认证（如需）

部分功能需要实名认证才能使用。如果系统提示需要认证：

1. 进入「个人中心」

2. 完成手机号验证

3. 按提示完成实名认证

新注册用户通常会赠送免费额度，可以直接使用 glm-4-flash 免费模型。
3

进入 API 密钥页面

登录后，直接打开 API 密钥管理页面：

https://open.bigmodel.cn/usercenter/apikeys

或者通过导航：点击右上角头像 → 「API 密钥」

页面顶部导航栏有「控制台」「财务」「个人中心」等入口，在「个人中心」下找到「API密钥」。
4

创建 API Key

在 API 密钥页面：

1. 点击 「创建 API Key」 按钮

2. 输入一个名称（如「OpenClaw」）

3. 点击 「确定」

API Key 只显示一次！创建后请立即复制保存。如果忘记了，需要重新创建一个。
5

复制 API Key

创建成功后，点击 API Key 旁的 「复制」 按钮。

API Key 格式类似：

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx.yyyyyyyyyyyyyyyyyyyy
建议把 API Key 临时保存到记事本中，方便稍后粘贴。
6

回到安装程序

复制好 API Key 后：

1. 切换回安装程序窗口（命令行黑色窗口）

2. 在提示处粘贴 API Key（右键粘贴或 Ctrl+V）

3. 按 Enter 确认

安装程序会自动测试 API 连接是否正常。

❓ 常见问题

Q: API Key 创建后看不到了？

A: API Key 只在创建时显示一次。如果丢失，请删除旧 Key 后重新创建。


Q: 提示余额不足？

A: 前往「费用中心」充值，或先使用免费的 glm-4-flash 模型。


Q: API 调用报错？

A: 检查 API Key 是否正确复制（注意首尾不要有多余空格），确认账户状态正常。
