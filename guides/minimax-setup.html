MiniMax — API Key 获取教程

性价比最高，MiniMax-M2.5 旗舰 230B MoE，SWE-Bench 80.2%

1

注册 MiniMax 开放平台账号

在另一个浏览器窗口中，打开 MiniMax 开放平台：

https://platform.minimax.io

点击 「注册」，使用手机号或邮箱完成注册。

已有账号则直接 「登录」。

2

实名认证（如需）

如果系统提示需要认证：

1. 进入「用户中心」→「基本信息」

2. 完成手机号验证和实名认证

MiniMax 新用户注册通常赠送免费额度，可以先体验再充值。
3

进入接口密钥页面

登录后，直接打开接口密钥管理页面：

https://platform.minimax.io/user-center/basic-information/interface-key

或者通过导航：用户中心 → 基本信息 → 「接口密钥」

4

创建 API Key

在接口密钥页面：

1. 点击 「创建新的密钥」

2. 输入名称（如「OpenClaw」）

3. 点击 「确认创建」

API Key 只显示一次！创建后请立即复制保存。
5

复制 API Key

创建成功后，点击 「复制」 按钮。

MiniMax API Key 格式类似：

eyJhbGciOiJSUzI1NiIsInR5cCI6IkpXVCJ9...
MiniMax 的 API Key 通常是一段较长的 JWT 格式字符串，请完整复制。
6

回到安装程序

复制好 API Key 后：

1. 切换回安装程序窗口（命令行黑色窗口）

2. 在提示处粘贴 API Key（右键粘贴或 Ctrl+V）

3. 按 Enter 确认

安装程序会自动测试 API 连接是否正常。

❓ 常见问题

Q: API Key 丢失了怎么办？

A: API Key 只在创建时显示一次。请在密钥管理页面删除旧 Key 后重新创建。


Q: highspeed 模型和普通模型有什么区别？

A: highspeed 模型输出速度更快（约 100 tokens/秒），效果基本一致，适合对响应速度有要求的场景。


Q: SWE-Bench 80.2% 是什么意思？

A: SWE-Bench 是代码编程能力的权威评测。80.2% 表示 MiniMax 在编程任务上表现非常出色。
